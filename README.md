# CoveyPlanner

7つの習慣 × プロジェクト管理 × Googleカレンダー連携

## 公開URL

```
https://SatoshiKishimoto.github.io/covey-planner/
```

## セットアップ手順

### 1. GitHubでリポジトリを新規作成
- リポジトリ名: `covey-planner`
- 公開設定: **Public**

### 2. ファイルをアップロード
- `index.html`（メインアプリ）をルートにアップロード

### 3. GitHub Pagesを有効化
Settings → Pages → Branch: **main** / `/(root)` → Save

数分後に公開されます。

### 4. GASプロキシを設定
アプリの「📆 カレンダー」タブからGASのURLを設定。
