Googleカレンダー連携設定
CORS制限を回避するため、Google Apps Script（GAS）を無料プロキシとして使用します。一度セットアップすれば複数カレンダーを完全無料で同期できます。
1

Google Apps Script を新規作成

script.google.com を開き、「新しいプロジェクト」をクリック。

2

下のコードを貼り付けて保存

既存コードを全削除し、下をコピーして貼り付け → Ctrl+S

function doGet(e) { var url = e.parameter.url; if (!url) return ContentService.createTextOutput('error: no url').setMimeType(ContentService.MimeType.TEXT); try { var res = UrlFetchApp.fetch(url, {muteHttpExceptions: true, followRedirects: true}); return ContentService.createTextOutput(res.getContentText('UTF-8')).setMimeType(ContentService.MimeType.TEXT); } catch(err) { return ContentService.createTextOutput('error: ' + err).setMimeType(ContentService.MimeType.TEXT); } }
3

ウェブアプリとしてデプロイ

右上「デプロイ」→「新しいデプロイ」→ 種類「ウェブアプリ」→ アクセス「全員」→「デプロイ」

⚠ 「アクセスを承認」が表示されたらGoogleアカウントでログインして許可してください。
4

デプロイURLを下に貼り付け → 保存

デプロイ後に表示される「ウェブアプリのURL」（https://script.google.com/macros/s/.../exec）を貼り付けます。

✅ このURLはあなた専用です。他人に教えなければ安全です。
